> 结构体默认的是 public，类默认的是 private。



## 一 内存结构



### 1 [内存对齐][1]

>  内存对齐可以理解为两步：变量对齐和整体对齐。
>
> 变量对齐是编译器的对齐系数（`#pragma pack(n)`）和成员大小最小的那个，即 `min（对齐系数，成员大小）`
>
> 整体对齐是整体大小能被最大的成员变量整除

假设在 32 位的系统上：

```c++
struct {
    int i;    	// 相对偏移 4 
    char c1;  	// 相对偏移 5 
    char c2;  	// 相对偏移 6
}x1;			// 相对偏移 8

struct {  
    char c1;  	// 相对偏移 1
    int i;  	// 相对偏移 4
    char c2;    // 相对偏移 9
}x2;			// 相对偏移 12
```
